HTML Fairy Tale Book

Creating a Card

Overview

Cards group related information. Use padding, shadows, and rounded corners for a magical effect.

<div class="card">
  <h2>Title</h2>
  <p>Description</p>
</div>

FAQ

Q: Can I have multiple cards in a row?
A: Wrap cards in a flex container:

<div class="cards-container">
  <div class="card">Card 1</div>
  <div class="card">Card 2</div>
</div>

Q: Can I use gradients inside a card?
A: Example:

.card {
  background: linear-gradient(135deg, #f3c677, #ff7f50);
}

Floating / Hover Cards

Overview

Hover effects can make cards lift and glow for a magical fairy-tale feel.

.card:hover {
  transform: translateY(-15px);
  box-shadow: 0 20px 30px rgba(243,198,119,0.5);
}

FAQ

Q: Can I make hover effects slower or faster?
A: Adjust the transition duration:

.card {
  transition: transform 0.7s ease, box-shadow 0.7s ease;
}

Q: Will hover work on mobile?
A: Use :active for touch feedback:

.card:active {
  transform: translateY(-15px);
}

Gradients

Overview

Gradients make elements colorful and magical. Try multiple colors.

background: linear-gradient(145deg, #2b2b3f, #1f1f2c);

FAQ

Q: Can I animate gradients?
A: Yes! Example:

@keyframes gradientMove {
  0% { background-position: 0% 50%; }
  100% { background-position: 100% 50%; }
}
.card {
  background: linear-gradient(270deg,#f3c677,#ff7f50,#9b59b6,#3498db);
  background-size: 600% 600%;
  animation: gradientMove 10s ease infinite;
}

Text & Code

Overview

Use <p> for text and <pre><code> for code blocks.

<p>Hello world</p>
<pre><code>console.log("Hi");</code></pre>

FAQ

Q: Can I style <code> blocks?
A: Example:

code {
  background: rgba(255,255,255,0.1);
  padding: 3px 5px;
  border-radius: 5px;
}

Images

Overview

Images bring your page to life. Always use alt text.

<img src="https://via.placeholder.com/150" alt="Example">

FAQ

Q: Resize images with CSS?
A: Example:

img {
  max-width: 100%;

}

Audio

Overview

Embed audio using <audio> with controls.

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
</audio>

FAQ

Q: Can audio autoplay?
A: Example:

<audio autoplay muted controls>
  <source src="audio.mp3" type="audio/mpeg">
</audio>

Forms & Collecting Data

Overview

Forms let users input information. Use JavaScript to collect it.

<form id="fullForm">
  <input type="checkbox" name="hobby" value="Reading">Reading<br>
  <input type="radio" name="gender" value="Male">Male<br>
  <select id="country"><option>USA</option></select>
  <textarea id="message"></textarea>
  <button type="submit">Submit</button>
</form>

FAQ

Q: How to collect form data?
A: Example using JavaScript:

const fullForm = document.getElementById('fullForm');
fullForm.addEventListener('submit', function(event){
  event.preventDefault();
  const hobbies = Array.from(document.querySelectorAll('input[name="hobby"]:checked')).map(e => e.value);
  const gender = document.querySelector('input[name="gender"]:checked')?.value;
  const country = document.getElementById('country').value;
  const message = document.getElementById('message').value;
  console.log({hobbies, gender, country, message});
});
